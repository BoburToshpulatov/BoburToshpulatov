# Featured Projects

## 🧠 Immersive Anatomy

Immersive anatomy learning experience built for Apple Vision Pro.

The project explores how Spatial Computing can transform anatomy education by allowing students to interact with anatomical structures in a natural three-dimensional environment.

### Technologies

`SwiftUI` • `RealityKit` • `visionOS`

🔗 Repository

https://github.com/BoburToshpulatov/Anatomy-Atlas-VisionPro

---

## 📚 Lumi – A Journey Through Fables

Educational sequencing game designed for children.

Players reconstruct fairy tales by arranging story events in the correct order, helping develop logical thinking, storytelling skills, and comprehension.

Published on the App Store.

### Technologies

`SwiftUI` • `AVFoundation`

📱 App Store

https://apps.apple.com/us/app/lumi-a-journey-through-fables/id6773034104

---

## 🏠 Clavem

Property management application developed at the Apple Developer Academy.

Designed to simplify property administration, tenant management, expenses, reminders, and daily workflows for landlords.

Published on the App Store.

### Technologies

`SwiftUI` • `SwiftData`

📱 App Store

https://apps.apple.com/it/app/clavem/id6759971969

🔗 Repository

https://github.com/BoburToshpulatov/Challenge5-CLAVEM
